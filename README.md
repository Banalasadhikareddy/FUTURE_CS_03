# FUTURE_CS_03
Task 3 Secure File Sharing System

This project is a **Secure File Sharing System** developed as part of **Task 3**.  
It allows users to upload files securely, encrypts them using **AES encryption**, and stores them safely.  
Users can also download files which are decrypted securely.
